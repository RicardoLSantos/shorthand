CodeSystem: iOS Physical Activity CodeSystem

Official URL: https://2rdoc.pt/ig/ios-lifestyle-medicine/CodeSystem/ios-physical-activity-cs Version: 0.1.0
Active as of 2025-11-22 Computable Name: IOSPhysicalActivityCS

CodeSystem for physical activity types from Apple HealthKit HKWorkoutActivityType. Enables interoperability between iOS Health App data and standard terminologies (SNOMED CT, LOINC).

Represents physical activity types captured by iOS Health App (HKWorkoutActivityType). Used as source codes in ConceptMaps to SNOMED CT for semantic interoperability with clinical systems.

This Code system is referenced in the content logical definition of the following value sets:

  • Este CodeSystem não é utilizado aqui; pode ser utilizado noutro local (por exemplo, em especificações e/ou implementações que utilizem este conteúdo)
